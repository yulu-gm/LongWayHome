# 场景设计强制参考

> 来源：用户在 2026-05-15 提供的第二张参考图。
> 用途：当未来会话无法直接看到原图时，本文件作为强制视觉参考摘要。所有影响画面、场景、车辆、UI、天气、灯光、道路、POI、资产密度的任务都必须参考本文件。

---

## 1. 总体视觉目标

关键词：

```text
低多边形 + 像素光照 + 柔和色彩 + 电影感氛围 + 2.5D 俯视公路旅行
```

画面不应是纯调试沙盒，也不应像空旷玩具模型。目标是让玩家一眼感到：

- 这是一段正在继续的长途旅行。
- 道路周围有可探索、可停靠、可发生事件的空间。
- 不同地区、天气和时间真的改变了驾驶情绪。
- 车辆是玩家的家，车顶行李、后备箱和车内布置都在讲述旅途。

---

## 2. 场景设计参考

### 郊外公路（白天 / 晴）

- 视野开阔，适合长途巡航。
- 道路应有清晰弯道、黄色中心线、路肩、护栏或电线杆。
- 周围应有河流、松树、岩石、草坡、远山。
- 事件密度较少，重点是放松驾驶和风景。

### 小镇街道（傍晚）

- 商店、加油站、餐馆、任务和事件集中区域。
- 道路较复杂，有十字路口、人行横道、路灯、招牌、停靠车辆。
- 光色偏暖，建筑窗户和招牌形成旅行中的落脚感。

### 高速公路（夜晚 / 雨天）

- 车流较快，需要专注驾驶。
- 路面应有湿润反光、车灯、尾灯、路灯和道路标识。
- 画面色调偏蓝黑，光源形成长条反射。
- 能见度较低，但车道线和前车尾灯必须清楚。

### 山区道路（白天 / 多云）

- 弯道多、地形起伏、天气变化影响大。
- 需要岩石、松树、山坡、护栏、坡度感。
- 适合表现车辆重量、抓地、悬挂和路面风险。

### 沙漠荒野（黄昏）

- 补给稀少，长距离驾驶，孤独感更强。
- 需要暖橙色天空、沙尘、仙人掌、荒地、稀疏加油站。
- 道路可以更直、更长、更空。

### 城市区域（夜晚）

- 路况复杂，行人车辆多，容易剐蹭。
- 需要霓虹、红绿灯、密集建筑、湿路反光、路口车流。
- 不是 MVP 第一优先，但要作为后续高密度区域方向。

## 3. 车辆设计参考

车辆必须通过轮廓、载物方式和驾驶手感区分。

### 老旧旅行车

- 最适合主角车。
- 平衡型，储物空间大，适合长途旅行。
- 视觉特征：旧车身、车顶行李、备用轮胎、旅行箱、轻微磨损。

### 皮卡

- 载重大、耐久强，但油耗较大。
- 视觉特征：开放货斗、工具箱、越野感。

### 小面包车 / Van

- 空间大，可改装为移动小屋，速度较慢。
- 视觉特征：高车顶、宽车身、内部生活空间潜力。

### 运动轿车

- 速度快、操控好，但储物空间小、耐久较低。
- 视觉特征：低车身、红色或高饱和车漆。

### 越野车

- 适合任意地形，适合复杂地形和旧路探索，油耗较高。
- 视觉特征：车顶架、备胎、粗胎、护杠。

## 4. 改装与储物视觉参考

车辆改装应能直接从画面读出来。

推荐可视改装：

- 车顶行李架：增加储物。
- 备用轮胎：减少爆胎风险。
- 加强保险杠：降低碰撞损伤。
- 扩容油箱：增加续航。
- 车尾储物箱：增加细格。

储物空间参考：

- 后备箱：工具、箱包、油桶、备用零件。
- 车内储物：地图、磁带、水、咖啡、小挂件。

实现要求：

- 即使用低模占位，也要先保留“东西在车上”的轮廓。
- 车顶载物会强化“车是家”的核心体验。

## 5. 环境设计参考

### 路边环境细节

可组合资产：

- 护栏
- 限速牌
- 方向牌
- 路障牌
- 灌木
- 仙人掌
- 枯树
- 邮箱
- 垃圾桶
- 路锥
- 岩石
- 木箱
- 油桶
- 轮胎
- 工具箱

这些物件用于：

- 丰富道路边缘。
- 提供事件点和 POI 识别。
- 让道路不再像空白测试场。

### 可互动地点 / POI

MVP 优先：

- 汽车旅馆
- 加油站
- 修理厂
- 休息区 / 观景台
- 路边摊贩
- 废弃建筑

这些地点不需要一开始可进入室内，但外观必须能被玩家从驾驶视角认出。

## 6. 天气与时间参考

参考图给出了天气与时间矩阵：

- 时间：白天、傍晚、夜晚、深夜
- 天气：晴天、雨天、雾天、雪天

MVP 可以先做：

- 白天晴天
- 傍晚晴天
- 夜晚雨天
- 白天雾天

每种组合必须至少改变：

- 主光方向和颜色
- 环境色
- 路面明暗和反射
- 能见度
- 车灯/尾灯可见性
- HUD 天气状态

不要只改 UI 数字。

---

## 7. 环境氛围参考

氛围目标：

- 雨夜海岸/湖边道路：湿路、尾灯、冷蓝色、孤独感。
- 驾驶舱：方向盘、仪表、收音机、小挂件，强化“车内生活”。
- 汽车旅馆房间：暖光、床、海报、窗外夜色，作为旅途休息点。

这些可以后置，但在做 UI、音频、休息站和车内事件时必须保留方向。

---

## 8. 颜色与材质方向

整体避免过度鲜艳或纯写实。

建议：

- 公路：灰黑沥青，黄色/白色车道线。
- 森林：深绿、灰绿、棕色岩石。
- 沙漠：暖橙、土黄、低饱和红棕。
- 夜雨：蓝黑、冷灰、红色尾灯、黄色路灯。
- UI：深色半透明底、浅色文字、琥珀色重点信息。

2. 是否强化了 2.5D 公路旅行氛围？

4. 是否仍然只是调试占位？如果是，缺少哪些资产？
5. 是否需要更新 `docs/road_trip_visual_asset_gaps.md`？
